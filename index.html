Projet universitaire 1 : Page web type marketplace

Ce projet a été réalisé en groupe de trois avec mes camarades. Nous avons utilisé uniquement du HTML et du CSS. L’objectif était de créer une page web de type marketplace, inspirée de sites comme Adidas ou Louis Vuitton. Projet universitaire 2 : Jeu en lien avec la gestion

Dans le cadre du cours de gestion-économie, il nous a été demandé de créer un jeu vidéo à la fois ludique et sérieux, permettant d’apprendre des notions de gestion ou d’économie. Avec mon groupe, nous avons choisi de développer un jeu en Python en utilisant Pygame. Il s’agit d’un jeu de type flashcards, avec des points gagnés à chaque bonne réponse et un système de score.
